### 定义

解决两个软件实体间的接口不兼容问题，不需要改变已有的接口，就能够使它们协同作用

### 使用场景

电源适配器

### 实现

提供适配方法，兼容多种接口

```javascript
var googleMap = {
  show: function () {
    console.log('开始渲染谷歌地图)  
  }  
}

var baiduMap = {
  display: function () {
    console.log('开始渲染百度地图')  
  }  
}

var baiduMapAdapter = {
  show: function () {
    return baiduMap.display()  
  }
}

renderMap(googleMap)
renderMap(baiduMapAdapter)
```
